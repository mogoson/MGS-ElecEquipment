  [Summary]
    Unity plugin for make electronic component in scene.
--------------------------------------------------------------------------
  [Environment]
    Unity 5.0 or above.
    .Net Framework 3.5 or above.
--------------------------------------------------------------------------
  [Achieve]
    Button : Button switch, self lock switch.
    Knob : Knob switch.
    Rocker : Rocker handle.
    LED : Simple LED light.
--------------------------------------------------------------------------
  [Usage]
    Reference the prefabs and demos to use the components in your project.
--------------------------------------------------------------------------
  [Demo]
    Prefabs in the path "MGS-ElectronicComponent/Prefabs" provide reference to you.
    Demos in the path "MGS-ElectronicComponent/Scenes" provide reference to you.
